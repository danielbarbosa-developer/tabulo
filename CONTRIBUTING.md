# Contributing to Tabulo

Thanks for your interest in contributing

Tabulo is currently in **beta**, so things may change quickly. 
Feedback, bug reports, and performance insights are especially valuable right now.

## Project principles

- Performance first
- Zero/low allocation mindset
- Simplicity over abstraction
- Explicit over magic

## Ways to contribute

- Reporting bugs
- Suggesting improvements
- Improving performance
- Adding tests
- Adding benchmarks
- Improving documentation

## Reporting issues

Please include:

- Clear description
- Sample CSV (if possible)
- Model used
- Expected vs actual behavior
- Minimal reproducible example

## Feature requests

Before suggesting a feature:

- Check if it aligns with Tabulo's goal: **high-performance CSV parsing**
- Avoid large abstractions or heavy features
- Prefer simple and composable solutions

## Pull requests

Please ensure:

- Code follows existing style
- No unnecessary allocations
- Performance impact is considered
- New behavior includes tests
